# Note-Taker
  ## Note taking application that allows saving and deleting of user created notes.

  This note taking application allows the user to save, inspect, and delete their own personal notes or reminders. Its allows for user input on the right hand column and stores any saved notes on the left hand column. If the user wishes     to inspect a note, clicking any saved note from the left hand column will populate the specified note to the right. The application also allows for a clear all button when the user inputs anything inside of the title or body of the        form. Finally, any stored notes have a delete functionality that removes any unwanted notes individually.
